# I2C Master and Slave Controller (i2cctrl42)

## Overview
This IP core implements the physical and data link layers of the standard I2C (Inter-Integrated Circuit) two-wire serial communication protocol. This project includes both Master and Slave Verilog modules (`i2c_master.v` and `i2c_slave.v`), designed for seamless integration and mixed-signal co-simulation. 

The Master Core handles the generation of the Serial Clock (SCL), transaction initiation (START/STOP conditions), and addressing. It provides a `busy` flag to prevent data collision from the host system. The Slave Core monitors the bus for its unique 7-bit address and handles Acknowledgement (ACK/NACK) and data shifting, utilizing `rx_valid` and `tx_req` handshakes to communicate seamlessly with local memory or peripherals. Both cores are strictly constrained to 8-bit data buses to ensure clean digital-to-analog translation through ADC/DAC bridges in the eSim/Ngspice physics environment.

## Pinout / I-O Ports

### Master Core (`i2c_master.v`)
| Port Name     | Direction | Width | Description                                      |
|---------------|-----------|-------|--------------------------------------------------|
| clk, rst_n    |Input      | 1-bit | System clock and active-low reset                |
| start_tx      | Input     | 1-bit | Trigger to initiate I2C transaction              |
| master_tx_8b  | Input     | 8-bit | 8-bit data payload to transmit                   |
| master_rx_8b  | Output    | 8-bit | 8-bit data received from slave                   |
| busy          | Output    | 1-bit | HIGH when Master is actively controlling the bus |
| scl           | Output    | 1-bit | I2C Serial Clock (Generated by Master)           |
| sda           | InOut     | 1-bit | I2C Serial Data (Bidirectional)                  |

### Slave Core (`i2c_slave.v`)
| Port Name     | Direction | Width | Description                                       |
|---------------|-----------|-------|---------------------------------------------------|
| clk, rst_n    | Input     | 1-bit | System clock and active-low reset                 |
| slave_addr    | Input     | 7-bit | Configurable 7-bit device address                 |
| slave_tx_8b   | Input     | 8-bit | 8-bit data payload to transmit to Master          |
| slave_rx_8b   | Output    | 8-bit | 8-bit data received from Master                   |
| rx_valid      | Output    | 1-bit | Pulses HIGH when new data is ready in slave_rx_8b |
| tx_req        | Output    | 1-bit | Pulses HIGH when Master requests data             |
| scl           | Input     | 1-bit | I2C Serial Clock (Read from bus)                  |
| sda           | InOut     | 1-bit | I2C Serial Data (Bidirectional)                   |
