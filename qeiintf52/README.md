# Quadrature Encoder Interface (qeiintf52)

## Overview
The QEI module tracks the rotation and position of motor shafts by interpreting the Phase A, Phase B, and Index pulse signals generated by quadrature encoders. The digital logic utilizes an edge-detection circuit to monitor state transitions, allowing for 4x resolution tracking. Based on the transition sequence, a direction flag is updated, and a position counter is incremented or decremented, while the Index pulse acts as a zero-reference reset.

## Pinout / I-O Ports
| Port Name   | Direction | Width  | Description                    |
|-------------|-----------|--------|--------------------------------|
| clk, rst_n  | Input     | 1-bit  | System clock and reset         |
| phase_a     | Input     | 1-bit  | Quadrature Phase A input       |
| phase_b     | Input     | 1-bit  | Quadrature Phase B input       |
| index       | Input     | 1-bit  | Zero-reference pulse           |
| position    | Output    | 16-bit | Tracked shaft position counter |
| direction   | Output    | 1-bit  | Rotational direction flag      |

## Block Diagram

